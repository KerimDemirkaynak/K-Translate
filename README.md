## 🌟 Overview

**K-Translate** is a powerful browser extension designed to break down language barriers without cluttering your web browsing experience. 

Unlike traditional translation extensions that alter the website's original aesthetics or constantly distract you with persistent floating buttons, K-Translate features an intelligent **Ghost UI**. The interface remains completely hidden on pages that are already in your preferred language or listed on your blacklist, appearing seamlessly only when foreign text is detected and needs translation.

## ✨ Key Features

* **Smart Ghost UI:** The translation panel and floating action button stay hidden on native or blacklisted pages, keeping out of your way until you actually need it.
* **Flawless Layout Preservation:** Powered by a robust HTML masking engine, it translates live text while keeping the website's original structure, fonts, inline tags (bolding, custom styling), and links completely intact.
* **Advanced Blacklist Management:** Quickly exclude specific domains or target languages from translation directly via the action panel ("Never translate this site" / "Never translate this language").
* **Auto-Detect & Manual Adjustments:** Automatically scans and detects the source language, converting it directly to your preferred language. You can also manually swap source/target choices on the fly.
* **Instant Revert:** Confused by a phrase? Switch back to the original text layout instantly with a single click.
* **High Efficiency:** Processes domestic DOM text blocks in smart, throttled batches to ensure lightning-fast execution without heavy browser memory footprints.
